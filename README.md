# Eden's Flower Garden
A cozy gardening game where you help Eden the Frog grow a beautiful flower garden

## Current Features:
- Welcome Screen
- New Game / Load Game System
- Main Menu System
- Game Loop
- Coin System
- 5 Slot Garden
- Planting Flowers:
-     Daisy
-     Tulip
-     Rose
-     Bellflower
-     Sunflower
-     Lavender
-     Hibiscus
- Flower Growth System:
-     Seed
-     Sprout
-     Bloomed
- Day Progression by Sleeping
- Harvest Bloomed Flowers
- Coin Reward System
- Visit Choco the Chipmunk's Flower Shop
  - Purchase Flower Seeds
  - Different seed costs
- Seed Inventory System
- Flower Journal:
  -Discover new flowers
  -Track harvest count
- Save and Load System:
  - Saves coins
  - Saves day progression
  - Saves state of garden
  - Saves inventory
  - Saves flower discoveries
- Object-Oriented Flower Data Using A Custom Flower Struct
