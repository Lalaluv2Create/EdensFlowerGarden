# Eden's Flower Garden
A cozy gardening game where you help Eden the Frog grow a beautiful flower garden

## Current Features:
- Welcome Screen
- Meet Eden the Frog
- Main Menu System
- Planting Flowers:
-     Daisy
-     Tulip
-     Rose
-     Bellflower
-     Sunflower
-     Lavender
-     Hibiscus
- Flower Growth System:
-     Seed
-     Sprout
-     Bloomed
- Day Progression by Sleeping
- Harvest Fully Grown Flowers
- Coin Reward System
- Visit Choco the Chipmunk's Flower Shop
- Purchase Flower Seeds
- Seed Inventory System
- 5 Slot Garden
- Object-Oriented Flower Data Using A Custom Flower Struct 

## Built with:
- c++
- g++
- Notepadd++
- GitHub

This project is a part of my video game development portfolio while studying Computer Science and Studio Art. My goal is to practice c++, object-oriented programming, and game design by building a cozy game from the ground up before recreating it in Godot.
