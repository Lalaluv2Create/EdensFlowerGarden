# Eden's Flower Garden

A cozy gardening game where you help Eden the Frog grow a beautiful flower garden

## Current Features:
- Welcome Screen
- Main Menu System
- Game Loop
- Coin System
- 5-slot garden
- Planting Flowers:
-     Daisy
-     Tulip
-     Rose
-     Bellflower
-     Sunflower
-     Lavender
-     Hibiscus

## Built with:
- c++
- Notepadd++
- GitHub
